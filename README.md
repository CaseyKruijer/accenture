# Holiday Assessment Backend

This project is a backend application created for the Accenture Holiday Assessment. It is built using Spring Boot and provides REST APIs documented with Swagger.

# Prerequisites
* Java 21 or higher
* Maven 3.9+
* An IDE such as IntelliJ IDEA
# Setup Instructions
1. Clone the repository to your local machine:
```bash
   git clone https://github.com/CaseyKruijer/accenture.git
```
3. Open the project in your IDE (e.g., IntelliJ IDEA).
4. Navigate to the project root folder and install dependencies:
```bash
   mvn clean install
```
5. Run the Spring Boot application:
```bash
   mvn spring-boot:run
```   
# API Documentation

Once the application is running, you can access the Swagger UI here:

http://localhost:8080/swagger-ui/index.html#

From this page, you can explore and test all available endpoints.
